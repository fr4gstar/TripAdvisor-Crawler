# TripAdvisor-Crawler

### Database

Configure your own database environment settings in php scripts.

### Develop mode

```bash
npm start
```

### Production mode
1. Use "systemjs.config.server.js" in index.html
2. Enable production mode
```angular2
 enableProdMode();
```

### Documenation with CompoDoc
```bash
npm run compodoc
```
